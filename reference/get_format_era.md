# Get format era for a given year

Returns the format era identifier for a given school year. Wyoming has
two major format eras:

- "pdf": 2000-2007 (PDF files on edu.wyoming.gov)

- "modern": 2008-present (reporting.edu.wyo.gov interactive reports)

## Usage

``` r
get_format_era(end_year)
```

## Arguments

- end_year:

  School year end

## Value

Character string indicating the format era
